# smart_splash_kit_example

Demonstrates how to use the `smart_splash_kit` plugin with 5 ready-to-use templates:

1. **Simple Logo Splash** – Centered logo with fade animation
2. **Animated Logo Splash** – Logo + text with fadeScale animation
3. **Loader Splash** – API simulation with retry on error
4. **Branding Splash** – Gradient background with skip button
5. **Config-Based Splash** – JSON-driven splash with performance monitoring

## Running the Example

```bash
cd example
flutter pub get
flutter run
```
